# dtr2epf

GUI-приложение на Rust для сборки модуля внешней обработки 1С из исходников Datareon Platform.

Интерфейс использует CPU software renderer и не требует OpenGL, DirectX 12 или дискретной видеокарты.

## Возможности

- загружает информацию об обработчиках 1С из выгрузки Datareon Platform;
- формирует `ObjectModule.bsl` по шаблонам для подстановки в обработку 1С для отладки или синтаксического контроля;
- позволяет редактировать, импортировать и экспортировать шаблоны.

## Запуск и сборка

```powershell
cargo run
cargo build --release
```

Обычный запуск открывает приложение без выбранного каталога. Чтобы сразу загрузить определённые исходники, используйте ключ:

```powershell
dtr2epf.exe --source "D:\Datareon\project\src"
```

Исполняемый файл после release-сборки: `target\release\dtr2epf.exe`.

Запустить автоматические тесты:

```powershell
cargo test
```

## Генерация

1. Выберите каталог `src` конфигурации Datareon кнопкой «Обзор…» — дерево загрузится сразу. Путь также можно ввести вручную и нажать «Загрузить».
2. Выберите адаптеры или отдельные строки в дереве.
3. Выберите вариант генерации: «Для отладки» или «Для синтаксического контроля». По умолчанию используется вариант «Для отладки».
4. При необходимости измените шаблоны и имена областей.
5. Нажмите «Сформировать обработку» и сохраните `ObjectModule.bsl`.
6. Поместите модуль в исходники внешней обработки, соберите EPF и запустите требуемые синтаксические контроли/линтеры 1С.

## Плейсхолдеры

Шаблон модуля использует `{{FROM_PLATFORM}}`, `{{TO_PLATFORM}}`, `{{FUNCTIONS}}`, `{{REGION_*}}`, `{{GENERATED_AT}}` и `{{SELECTED_COUNT}}`.

Шаблоны методов используют `{{NAME}}`, `{{ORIGINAL_NAME}}`, `{{PARAMETERS}}`, `{{CODE}}`, `{{TYPE}}`, `{{INTEGRATION}}`, `{{ENTITY_ID}}`, `{{SOURCE_PATH}}`, `{{SUBSCRIPTION_OBJECT}}` и `{{SUBSCRIPTION_OBJECT_MANAGER}}`.

`{{SUBSCRIPTION_OBJECT}}` и `{{SUBSCRIPTION_OBJECT_MANAGER}}` заполняются только для обработчиков `Subscription1C / ToPlatform`. Например, для `Справочник.Автомобили` значением менеджера будет `Справочники.Автомобили`.
